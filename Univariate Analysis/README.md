Univariate Analysis – Data Science Assignment

This project demonstrates Univariate Analysis using Python and Pandas.

Files Included

Univariate.py – Contains functions for univariate statistical analysis.

univariate.ipynb – Jupyter Notebook used to call the functions and execute the analysis.

Functions Implemented

QuanQual() – Separates quantitative and qualitative variables.

uni_variate() – Calculates descriptive statistics (mean, median, mode, quartiles, IQR, skewness, kurtosis, variance, standard deviation).

CheckOutliers() – Detects outliers using the IQR method.

ReplacingOutlier() – Replaces outliers with boundary values.

FreqTable() – Creates frequency, relative frequency, and cumulative frequency tables.

Tools Used

Python

Pandas

Jupyter Notebook

This assignment demonstrates basic data preprocessing and statistical analysis techniques used in Data Science.
